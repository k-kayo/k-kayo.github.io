SKILL

Figma

デザインは主にFigmaで作成します。必要なバナーなどあればPhotoshopを使用します。アイコンやボタン、プラグインを使用した機能拡張、オートレイアウト機能、コンポーネント、データのインポートやエクスポートの機能など一通りの機能を使用してサイト制作します。

Photoshop

画像切り抜き、レタッチが可能です。ブラシツールでの描画、光彩やドロップシャドウ、彩度や明るさなどの調整が可能です。レイヤーマスクでオブジェクトを隠したり、見せたりなどの対応可能です。バナー作成はPhotoshopを使用します。

HTML/CSS

キーフレームアニメーションやトランジションを使用して、要素の移動、フェードイン/フェードアウト、回転などのアニメーションの制御、ドロワーメニューの作成可能です。またレスポンシブデザインも実装可能です。記述方法はsassになります。

JavaScript

主にjQueryライブラリー（slickやAOS）を用いた制作経験があります。
スライドやギャラリーで使用することが多いです。JavaScriptについては、今後も学んでいきたいと思います。

Illustrator

パスやシェイプを作成・編集が可能です。
ペンツールを使用して、直線、曲線、図形などを作成し、変形や結合、分割などの編集操作を行うことも可能です。

WordPress

独学では有料テーマを利用したブロックエディターでの制作経験がありま。PHP、FTP、カスタマイズについては、これから深く学んでいきたいと思っています。

ABOUT

初めまして、桑野 佳世と申します。

出身は大阪です。結婚を機に博多へ引っ越し、その後東京に住んで7年になります。
これまでの仕事では、主にお客様と接する業務に従事してきました。

■キャリアチェンジのきっかけについて：
数年前に、WordPressで個人ブログを立ち上げたことがきっかけで、WEBデザインに興味を持ち始めました。
その後、最愛の母を失ったことが大きな転機となり、「一度きりの人生、悔いのない人生を生きよう」と決心し、WEBデザインの学習に本格的に取り組むことを決断しました。
退職後は、都内の職業訓練校で半年間学び、その後もオンラインスクールなどを活用して、Webサイト制作やShopify構築のスキルを習得しました。

■WEBデザインからネットショップ運営への転身：
学んだWEBデザインを活かし、ECサイト専門のデザイナーとしてフリーランスを目指していましたが、過去の接客販売の経験から、「商品を扱うこと」や「顧客との関わり」に強い魅力を感じていたため、自然と「自分でネットショップを運営したい」という思いに変化していきました。
現在はレディースファッションのネットショップを立ち上げ、SNSのリール動画で商品の魅力を発信しています。
運営開始からまだ1ヶ月も経たず、広告運用もこれから開始する段階のため、現時点ではフォロワー数やリーチは少なく、売上もまだありません。しかし、今後はこれらの数字を伸ばすために、広告運用やコンテンツ戦略を強化していきたいと考えています。

■今後の目標について：
現在は、InstagramなどSNSを活用したプロモーションに力を入れています。リール動画の企画から編集、投稿までを自分で行い、より多くのフォロワーに響くコンテンツ作りに努めています。
私の目標は、ユーザーにとって最高のショッピング体験を提供することです。そのためにも、SNS動画編集スキルをさらに高め、より多くの方にブランドや商品の魅力を感じていただけるようにと考えています。

■自分自身の性格について：
明るく、楽しくを大切にする性格です。笑顔を大事にし、食事の時間も心から楽しんでいます。友人や知人からは「優しいね」や「面白い話をしてくれる」と言われ、職場では「真面目だね」と評価されることもあります。
家族との時間を大切にし、困ったときには手を差し伸べることを心がけています。 神仏を大切にすることで、心が落ち着く瞬間を大事にしています。時折、神社仏閣にお参りに行き、心をリセットしています。
写真は普段の自分らしさが出ているものを選びました。埼玉県秩父の三峯神社に夫婦で参拝した際の写真です。

■趣味について：
普段の息抜きとして、YouTubeでドキュメンタリーや学習系の動画をよく見ています。ECサイトのトレンドにも興味があり、日々の情報を集めています。
音楽鑑賞も楽しんでおり、邦楽・洋楽問わず様々なジャンルを聴きます。休日には神社仏閣へお参りに行ったり、リフレッシュしたい時は温泉で気分転換をしています。また、健康維持のために週に3回ほどウォーキングをし、街を散歩することを楽しんでいます。
