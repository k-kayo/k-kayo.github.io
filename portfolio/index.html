SKILL

HTML5/CSS3

キーフレームアニメーションやトランジションを使用して、要素の移動、フェードイン/フェードアウト、回転などのアニメーションの制御、ドロワーメニューの作成も可能です。またレスポンシブデザインも実装可能です。学んだ期間がPhotoshop同様に長いです。sassでの記述も可能です。

JavaScript

主にjQueryライブラリー（slickやAOS）を利用した制作経験があります。JavaScriptについては今後も深く学んでいきたいと思っています。

Photoshop

基本的な画像切り抜き、レタッチが可能です。ブラシツールでの描画、レイヤースタイルで光彩やドロップシャドウの調整、彩度や明るさは調整レイヤーを主に使用しています。レイヤーマスクでオブジェクトを隠したり、見せたりなども可能です。バナー作成にはPhotoshopを使うことが多いです。

Illustrator

パスとシェイプを作成し、編集することができます。ペンツールや形状ツールを使用して、直線、曲線、図形などを作成し、変形や結合、分割などの編集操作を行うことができます。アイコンやロゴはIllustratorで制作しております。

WordPress

独学では有料テーマを利用したブロックエディターでの制作経験のみでした。現在本格的に訓練校でテーマのカスタマイズを学んでいます。PHP、FTPソフトも同様に学んでいます。プラグインでの機能拡張、お問い合わせフォームの作成、ギャラリー表示、ソーシャルメディアとの連携など追加可能です。

Figma

アイコンやボタンのデザイン、レイアウトの作成など、さまざまなUI要素をデザインすることができます。プラグインを使用して機能を拡張し、オートレイアウト機能やコンポーネント機能も使え、データのインポートやエクスポートも行えます。デザインカンプはFigmaで行っています。

ABOUT

初めまして。webデザイナー見習いの桑野と申します。
大阪出身。結婚を機に博多へ引っ越し、東京に住み7年目になります。
これまでは主にお客様と接する仕事を経験してきました。

三年ほど前に個人ブログをWordPressで作成したことがキッカケでWEBデザインに興味を持ち、2023年2月から在職しながらUdemyやデイトラ等の動画学習で学んできました。本格的に学ぶため前職を退職し、2023年10月から2024年4月まで都内の職業訓練校にてWEBデザインを学びました。

現在はECサイトのデザイン制作、運営、管理、更新等に携われることを目標に日々邁進しております。EC業界に興味を持った理由は、ECサイトを普段から利用しており、出品側としてもメルカリで出品経験があったことが主な理由です。また過去の仕事で服飾販売の経験があったこともEC業界で仕事がしたいと思うようになった理由です。

自分自身の性格については、明るいもしくは面白い部類の人間かと思います。よく笑い、よく食べます。周りの方々からは優しい、面白い、真面目と言われることもあります。写真は普段の自分らしさが出ているものを選びました。

趣味は、音楽鑑賞（Spotify）、YouTube鑑賞（WEBデザインに関する動画、海外の方が日本料理や観光を楽しむチャンネル、K-popの動画等）を見ることが多く、普段の息抜きです。休日に外出の際は観光で神社仏閣へお参りに出掛け、御朱印を頂いたりするのも趣味です。リフレッシュしたい時はスパや温泉、ジムなどで汗を流して気分転換しています。
