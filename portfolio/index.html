SKILL

HTML/CSS

キーフレームアニメーションやトランジションを使用して、要素の移動、フェードイン/フェードアウト、回転などのアニメーションの制御、ドロワーメニューの作成可能です。またレスポンシブデザインも実装可能です。記述方法はsassになります。

JavaScript

主にjQueryライブラリー（slickやAOS）を用いた制作経験があります。
スライドやギャラリーで使用することが多いです。JavaScriptについては、今後も学んでいきたいと思います。

Photoshop

画像切り抜き、レタッチが可能です。ブラシツールでの描画、光彩やドロップシャドウ、彩度や明るさなどの調整が可能です。レイヤーマスクでオブジェクトを隠したり、見せたりなどの対応可能です。バナー作成はPhotoshopを使用します。

Illustrator

パスやシェイプを作成・編集が可能です。
ペンツールを使用して、直線、曲線、図形などを作成し、変形や結合、分割などの編集操作を行うことも可能です。

WordPress

独学では有料テーマを利用したブロックエディターでの制作経験がありま。PHP、FTP、カスタマイズについては、これから深く学んでいきたいと思っています。

Figma

アイコンやボタン、レイアウトなど、さまざまなUI要素をデザインすることができます。プラグインを使用した機能拡張、オートレイアウト機能、コンポーネント、データのインポートやエクスポートの機能など一通りの機能を使用してサイト制作します。デザイン制作はFigmaで行います。

ABOUT

初めまして。webデザイナーの桑野と申します。

大阪出身。結婚を機に博多へ引っ越し、東京に住み7年目になります。
これまでの仕事では、主にお客様に接する仕事を経験してきました。

webデザイナーになった経緯：
WordPressで個人ブログを作成したことからWEBデザインに興味を持ち始めました。ただ正直年齢的に難しいとも考え、半ば諦めて趣味程度にしようと当時は考えていました。
その矢先、親友のように仲の良かった最愛の母が突然脳卒中で倒れました。病院の検査で末期の肝臓がんと判明。頑張ってくれましたが、余命宣告通り半年で母は旅立ちました。
生まれた時から居て当たり前の存在だった母を失い、その時から人生観が変わりました。
人は皆限られた時間しか生きられない…。
自分は悔いのない人生を送れているだろうかと。
母から貰った命。これからはやりたい事をして生きていこうと決意しました。
それからは在職しつつオンライン学習で学んでいましたが、本格的に学ぶため前職を退職。都内の職業訓練校で半年間WEBデザインを学んできました。

現在について：
職業訓練校を卒業後、フリーランスとしてチャレンジしています。
現在はWEBデザイナーとして、スキルアップと営業活動に励んでいます。

今後の目標について：
ECサイトのデザイン制作、運営、管理、更新等に携われることを目標にしています。
理由は、ECモールを普段から利用していることもありますが、出品側としてもメルカリで出品・販売した経験があることや、また過去に服飾販売の経験があったことなどが理由です。

自分自身の性格について：
明るい、もしくは面白い部類の人間かと思います。よく笑い、よく食べます。周りの方々からは優しい、面白い、真面目と仰って頂くことがあります。
自分自身としては、面倒見が良い方で、親・家族想いで、神仏を大事にする心を持つ素直な性格かと思います。又分け隔てなく人と接する自分の性格が一番好きです。

趣味について：
音楽鑑賞（Spotify）、YouTube鑑賞（WEBデザインに関する動画、海外の方が日本料理や観光を楽しむチャンネル、K-popの動画等）を見ることが多く、普段の息抜きです。休日に外出の際は観光で神社仏閣へお参りに出掛け、御朱印を頂いたりするのも趣味です。リフレッシュしたい時はスパや温泉、ジムなどで汗を流して気分転換しています。
