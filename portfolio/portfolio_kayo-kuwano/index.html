SKILL

HTML5/CSS3

scssでの記述、キーフレームアニメーションやトランジションを使用して、要素の移動、フェードイン/フェードアウト、回転などのアニメーションを制御。ドロワーメニューの作成も可能です。レスポンシブデザインの実装。学んだ期間がPhotoshop同様に長いです。

JavaScript

主にjQueryライブラリー（slickやAOS）を利用した制作経験があります。JavaScriptについては今後も深く学んでいきたいと思っています。

Photoshop

基本的な画像切り抜き、レタッチが可能です。ブラシツールでの描画、レイヤースタイルで光彩やドロップシャドウの調整、彩度や明るさは調整レイヤーを主に使用しています。レイヤーマスクでオブジェクトを隠したり、見せたりなども可能です。バナー制作はPhotoshopが多いです。

Illustrator

パスとシェイプを作成し、編集することができます。ペンツールや形状ツールを使用して、直線、曲線、図形などを作成し、変形や結合、分割などの編集操作を行うことができます。アイコンやロゴはIllustratorで制作しております。

WordPress

独学では有料テーマを利用したブロックエディターでの制作経験のみでした。現在本格的に訓練校でテーマのカスタマイズを学んでいます。PHP、FTPソフトも同様に学んでいます。プラグインでの機能拡張、お問い合わせフォームの作成、ギャラリー表示、ソーシャルメディアとの連携など追加可能です。

Figma

アイコンやボタンのデザイン、レイアウトの作成など、さまざまなUI要素をデザインすることができます。プラグインを使用して機能を拡張し、デザインの自動化、データのインポートやエクスポートも行えます。デザインカンプはFigmaで行っています。

ABOUT

大阪出身ですが結婚を機に博多へ引っ越し、現在は都内在住です。
これまでは主にお客様と接する仕事に従事してきました。

三年ほど前に独学で個人ブログをWordPressで作成したことがキッカケとなりWEBデザインに興味を持ちました。深く学びたい気持ちと、仕事に出来たらという思いが強くなり、WEBデザインを仕事にしようと決心し、2023年2月から在職しながらUdemyやデイトラ等の動画学習で学んでいました。

本格的に学びたい気持ちが更に増し、2023年10月から職業訓練校で学ぶため前職を退職。
プライベートでECサイトを利用する機会も多く、又メルカリで自分自身が出品した経験があったり、過去の仕事で接客販売の経験があったことも重なり、webデザインの仕事を通じてお客様に喜んで頂けるEコマースの業界でデザインの仕事がしたいと思うようになりました。現在はECサイト及びShopifyでのサイト制作、運営、管理、更新等に携われることを目標に日々邁進しております。

自分自身の性格については、明るいもしくは面白い部類の人間かと思います。よく笑い、よく食べます。
趣味は、音楽鑑賞（Spotifyで主にK-popや洋楽を聴いています）、YouTube動画（WEBデザインに関する動画、海外の方が日本料理や観光を楽しむ動画、K-popの動画等）を見ることが普段の息抜きです。 最近は勉強に追われて行けていないのですが、スパや温泉、ジム、又美味しい料理も好きです。神社仏閣へお参りに行き、御朱印を頂くことも趣味の一つです。
